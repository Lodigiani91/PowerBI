# PowerBI
PowerBI 


Obiettivo del Progetto:

Questo progetto ha l’obiettivo di realizzare un report interattivo pensato per supportare i manager di TechMarket nelle attività di analisi e monitoraggio delle vendite.

Funzionalità principali

Il report consente di:

Visualizzare le vendite mensili, considerando sconto, unità vendute e prezzo.

Analizzare le vendite per città, per individuare trend e differenze territoriali.

Monitorare le performance di vendita per ogni prodotto, confrontando volumi e redditività.

Ottenere una panoramica completa dei dati per ciascun punto vendita, con indicatori chiave e metriche di performance.

Gestione dei Resi:

È inoltre inclusa una pagina dedicata ai resi, utile per:

Calcolare le vendite nette

Fornire una rappresentazione più accurata e realistica delle performance complessive
